# LearningFullter
Flutter学習用のリポジトリ

## Setup

### Android
1. Android Studioを入れておく
2. Android Studio内でFlutter Pluginをインストール
3. Preferences > Editor > Code Style > Dartからdartfmtを使用するようにチェックを入れておく

### iOS
1. Xcodeを入れておく

### Flutter SDK
1. https://flutter.io/docs/get-started/install/macos
    * これに沿っていろいろ設定
